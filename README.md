# Cebu Traffic Etiquette Website

A simple educational website designed to promote **better driving etiquette and road safety in Cebu City**.  
This project was created as part of a school initiative to raise awareness about responsible driving habits and traffic rules.

## 🌐 Project Overview

The website presents information about traffic etiquette in a clear and engaging way. It includes visuals, interactive elements, and a short quiz to help users better understand proper road behavior.

The goal is to make traffic safety **easy to understand and accessible**, especially for students and new drivers.

## ✨ Features

- 📊 Traffic etiquette infographic
- 📖 Information on basic road rules and driving etiquette
- 🧠 Interactive quiz to test knowledge
- 🎨 Simple, clean interface
- 📱 Responsive design

## 🛠️ Built With

- **HTML5** – structure of the website  
- **CSS3** – styling and layout  
- **JavaScript** – quiz functionality and interactivity  

## 🎯 Purpose

Traffic congestion and unsafe driving behaviors are common problems in many cities.  
This project aims to promote:

- Responsible driving
- Awareness of road rules
- Safer interactions between drivers, pedestrians, and cyclists
